# HR-Analytics-Employee-Attrition-Analysis
Project Overview
This project analyzes employee attrition patterns to identify the key factors influencing employee turnover. The dashboard provides insights into workforce distribution, job satisfaction, salary impact, promotions, and departmental trends to support data-driven HR decisions.

Project Objective
The main objective of this project is to:
Analyze employee attrition trends
Identify high-risk employee segments
Understand the impact of salary, promotion, job role, and satisfaction
Support HR in improving employee retention strategies

Tools Used
Python (Pandas, NumPy) – Data cleaning & analysis
Power BI – Data visualization & dashboard creation
CSV – Processed datasets for visualization

Dataset Features
The dataset includes employee-related information such as:
Age
Gender
Marital Status
Department
Job Role
Job Level
Job Satisfaction
Environment Satisfaction
Education Field
Salary Hike Percentage
Promotion Status
Total Working Years
Attrition (Yes/No)
